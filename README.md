This is my scraper for a cinema website, it uses a discord bot to output the content, but can just use the python file by running the function in the script.

This scraper find all the remaining screenings for the day(e.g. if it is 8am there will be more screenings left in the day than 5pm) and returns how busy these screens will be.

Mainly for use by cinema staff to be able to see how busy the day will be.
