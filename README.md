This is a automation script that goes to any arc cinema locations website,
visits each showing for today, and shows you how busy it will be.

This script shows the total number of screenings left in the day, the amount of seats
sold for all these screenings, the average number of seats sold per screen, and a 
statement on how busy it will be

Make sure to read the script and insert the relevent information where asked to do 
so, which is the bot token, and the arc cinema page link.
